Basic Concepts of Web Technology

Definitions

1 : Define HTMl: Html is defined as a markup language, it is used to create pages displayed on the web

2 : Define Client-server model of communications Client-server model represents commmunication between two computers over a network, clients request a resource and servers provide the resources.

3 : Write down the name of different types of servers Data server, Application server, Web server, Email server, Media server and Authentication server are all examples of a server type.

4 : Define a Web Server A web server is hardware and software that accepts incoming HTTP requests. A web browser starts the communication by requesting a resource from the web server, the web server then sends the resource to the web browser.

5 : Define Data Centers A facility that provides the computational power to run applications, multiple computers in the datacenter are networked together and are responsible for storing and proccessing data requried to run an application. Data centers provide failover redundancy in which a failure in one server will not impact performace as there are several servers to retain a site's availability.

Understanding

6: Why do we need a Uniform Resource Locator? To allow clients to request resources from a server, URL is a naming mechanism so that the client knows how to communicate with the server

7: What is the difference between a static and dynamic website? A static page consists of only HTML pages which are run of the client side, dynamic pages are created at run-time by a program running on a web server, the pages are converted to html by the program to be sent to the client.

8: Why do we need Domain Name System (DNS)? DNS allows for software to discover IP addresses from domain names and determine what the IP address of the server is to request resources from it and display a webpage.

9: Describe in detail the process of DNS address resolution? Address resolution starts with a client searching a URL in a web browser, if the site is not within the browser's memory it will then communicate with a DNS server to resolve the domain. If the DNS server does not contain the IP in its cache it will contact the root name server, which will return an IP of the name server. The primary DNS server then contacts the .com name servers requesting for an IP of name server for target domain. After this the primary DNS server now can contact the DNS server for the domain name and retrieve an IP address to be passed back to a clients DNS resolver. The client can not contact the IP address of the the domain name and request resources for web page display.

10: How can you pass a users information from a client to the web server? Query strings allow for passing information from client to server, they are encoded as key-value pairs and are preceeded with '?' symbol in a URL

11: What is the function of Hypertext Transfer Protocol ? HTTP is a protocol which allows for the communication between a client and a web server with the help of TCP to faciliate transmission control. HTTP has several different types of requests which specifiy what will happen with the content, for instance a GET request from client to server specifies that the client wants to connect to the server and get the .html resources and load the webpage.

12: Describe how a single web page is displayed on a web browser ? A web page is displayed by the client sending a GET request to the web server, the web server sends the appropriate file.html to the client. If the web page has styling and pictures the client will send more GET requests to retrieve these resources and show the web page in its entirety. Each part of the web page is sent individually; due to advancements in internet technology this process may not be seen and it will appear as the entire web page loaded in its entirety.

Examples

13: Give an example of Uniform Resource Locator ? Specify all components of an URL An example would be https://www.tradingview.com/chart/?symbol=KUCOIN%3ALUNCUSDT , where https represents the protocol, www.tradingview.com represents the domain, /chart/ is the path, and ?symbol=KUCOIN%3ALUNCUSDT represents the query string.

14: Give some examples of generic top-level domains (gTLD) ? An example of a generic top level domain would be .com , .org , .net

15: Give some examples of country code top-level domains (ccTLD) ? An example of a country code top level domain would be .ca (Canada), .us(United States), .uk (United Kingdom)

16: Give some examples of HTTP request methods. An example of an HTTP request method would be a GET request, from a client to a server asking for the html files to view a webpage. Another example would be a POST request, coming from a client to a server requesting to upload resources to the web site.
